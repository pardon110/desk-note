队列驱动

Laravel 为队列提供了的不同实现驱动：

    sync C 当任务被推送到sync队列后会立即执行
    database C 数据库队列会将任务存放到数据表中，该表默认是jobs
    beanstalkd C 要实现beanstalkd队列需要先配置并运行beanstalkd，而且还要使用Composer安装依赖包：composer require "pda/pheanstalk=~3.0"
    sqs C 该驱动会将任务推送到Amazon SQS队列，同样需要先使用Composer安装依赖包：composer require aws/aws-sdk-php
    iron C 该驱动会将任务推送到IronMQ，使用前需要安装依赖：composer required "iron-io/iron_mq=~1.5"
    redis C Redis队列会将任务推送到Redis数据库，使用前需要安装redis依赖：composer require "presdis/presis=~1.0"
